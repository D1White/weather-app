# Weather app ⛅

## Demo

[▶️ Live demo](https://1white-weather-app.netlify.app)

![](https://s8.gifyu.com/images/weather-app85ac5fc0fde1a505.png)


## Features:

* Switching temperature units.
* Weather for the week and every 3 hours.
* Temperature map.
* Search cities.
